What is Bridge Base?

Bridge Base points users to the current Ethereum-to-Base transfer routes for ETH and supported assets. Base's bridge documentation says the old bridge.base.org interface is deprecated and lists Superbridge and Brid.gg as Ethereum-to-Base options.

How Bridge Base works

  1. Open the routeUse the app button and verify the destination domain is superbridge.app before connecting a wallet.
  2. Connect walletConnect the wallet that holds the asset on Ethereum mainnet, then review the selected source and destination networks.
  3. Choose asset and amountSelect ETH or another supported token, confirm the recipient address, and check any wallet approvals carefully.
  4. Confirm and trackSubmit the transaction, then follow the bridge status in the app and verify the Base-side transaction when it lands.

Bridge Base FAQ

What is Bridge Base?

Bridge Base is a routing page for users who want to move assets between Ethereum and Base through a docs-listed bridge app; it is not operated by Coinbase or Base.

Does Bridge Base still use bridge.base.org?

No. Base documentation says the bridge on bridge.base.org has been deprecated and now points users toward Superchain bridge options.

What can I move with Bridge Base?

The Ethereum-to-Base route supports ETH and other assets supported by the selected bridge app. Always confirm token details in the app before signing.

How do I check a Bridge Base transaction?

Use the bridge app status, your wallet history, and BaseScan for the Base-side transaction record.

Is Bridge Base financial advice?

No. It is educational routing content; review bridge terms, smart contract risk, wallet prompts, and the destination domain before signing.
